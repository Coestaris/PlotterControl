CnC_WFA(CNCWFA)(CnC Windows Forms Aplication) - кодовое имя программы Plotter Control.

Устройство

 

Взаемодейсвтие с устройством

Управление устройством происходит через простой Serial Port, библиотека для работы такого есть фактически под любой язык. Стандартная скорость передачи данных (BDRATE) - 115200 бод, хотя ее можно заменить перепрошив контроллер с помощью стандартной Arduino IDE. Доступные команды управления вы можете найти тут.

Работа с памятью

Команды задачи памяти (процедуры) имеют следующий синтаксис: [command] + "значение" + ';', при этом, в случае успешной задачи, он вернет OK;, иначе FL;. Считывания: [command] + ';', при этом устройство отошлет по порту значение переменной.

Вид команды задачи выглядит таким образом: "set" + "имя переменной", о считывания "get" + "имя переменной". Список доступных переменных читайте тут.

Более подробно читайте на странице про "Прямые команды".

API для ПК

CWA - CNCWFA API. В данное API входят следующий набор .NET библиотек:

  1. CWA_Printing.DLL - содержит основные методы печати на устройстве. Более детально читайте тут.
  2. CWA_Vectors.DLL - содержит основные методы работы с векторами, векторизации. Более детально читайте тут.
  3. CWA_MathParser.DLL - содержит методы парсинга строки, как мат. выражения. Более детально читайте тут.
  4. CWA_VectorHM.DLL - содержит дополнительные методы работы с векторами. Более детально читайте тут.
  5. CWA_Connection.DLL - содержит основные программные, полуавтоматические методы работы с памятью устройства. Более детально читайте тут.
  6. CWA_ManualControl.DLL - содержит методы ручного управления устройством. Более детально читайте тут.
  7. CWA_VectEditorEnvironment.DLL - содержит методы для работы с веторными документами. Более детально читайте тут.
