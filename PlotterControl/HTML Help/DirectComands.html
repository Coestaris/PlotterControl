Прямые команды

Устройство контролируется с помощью команд. Их общий вид таков: [command];. ';'(точка с запятой) - обязательный символ, знак конца команды. Без него устройство не воспримит строку как команду.

Команды делятся на 2 типа:

  • Процедуры - команды, которые просто выполняются, не возвращая значений;
  • Функции - команды, которые выполняются, возвращая какое-то значение.

Большинство команд направлены на работу с памятью устройства, хотя основными являются 2 команды перемещения пера.

Прошивка работает с данными - переменными которые хранятся постоянной памяти EEPROM. Со списком и назначением их Вы можете ознакомится тут.

С полным списком доступных на данный момент команд вы можете ознакомится тут.

Пример взаемодейстия с устройством:

| [PC]: resetd;

| [Device]: OK;

| [PC]: setidle30; setwork20; setp293;

| [Device]: OK; OK; OK;

| [PC]: settte;

| [Device]: Unknown Command "settte";

// Отправка команды "Установки параметров по умолчанию";

// Отчет о удачном выполнении команды;

// Задача параметров IDLE = 30, WORK = 20 и P29 = 3;

// 3 ответа устройства о успехе;

// Неправильная команда

// Ответ устройства о неправильности команды
