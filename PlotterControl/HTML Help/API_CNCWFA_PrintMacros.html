Библиотека CWA_VectorHM

Перечисление MacrElemType

Описывает тип действия элемента. Содержит следующие варианты:

  • Tool - перемещение инструмента.
  • MoveToPoint - перемещение в конкретную точку.
  • MoveRelative - относительно смешение.
  • ToolAndDelay - перемещение инструмента с задержкой после.
  • Delay - задержка.
  • None - ничего не делать.
  • MoveToPointAndDelay - перемещение в конкретную точку с задержкой после.
  • MoveRelativeAndDelay - относительно смешение с задержкой после.

Перечисление MacrElemTypeShorted

Содержит сокращенные эквиваленты перечислению MacrElemType для сохранения. Содержит следующие варианты:

  • T = Tool.
  • MTP = MoveToPoint.
  • MR = MoveRelative.
  • D = Delay.
  • N = None.
  • TAD = ToolAndDelay.
  • MTPAD = MoveToPointAndDelay.
  • MRAD = MoveRelativeAndDelay.

Структура MacrElem

Содержит статические методы:

  • public static MacrElemTypeShorted NormalToShorted(MacrElemType t) - приведение обычного типа к короткому.
  • public static MacrElemType ShortedToNormal(MacrElemTypeShorted t) - приведение коротого типа к обычному.

Содержит следующие поля:

  • public MacrElemType Type - тип элемента.
  • public int ToolMove - количество шагов перемещения инструмента (вверх или вниз).
  • public System.Drawing.PointF MoveToPoint - перемещение в точку.
  • public System.Drawing.PointF MoveRelative - относительное смещение.
  • public int Delay - задержка в мс.
  • public string StringType - красивое описание данного элемента.
