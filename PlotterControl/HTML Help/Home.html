Plotter Control
Что оно такое и как куда его использовать?

Plotter Control - много функциональная программа для управления самодельными (аматорскими) устройствами типа ЧПУ, собранных на основе контроллера Arduino В ее, функционал, кроме построений, также входят:

  • Настройка, прошивка, задача параметров, калибровка устройства;
  • Создание и обработка векторных рисунков;
  • Создание последовательностей команд для устройства (макросов) и многое другое.
Имеет возможность построения:
  • Растровых изображений;
  • Веторных изображений формата .SVG;
  • Математических графиков, кривых.

С чего начать?

Базовый уровень
  • Если у вас еще нет устройства соберите его (По этой ссылке вы можете найти чертежи и схемы моего планшетного графопостроителя);
  • Получите необходимые программе параметры а вашем устройстве;
  • Настройте программу под себя;
  • Получите вектор:
    • Векторизацие растра;
    • С .SVG файла;
    • С математического выражения;
    • Или нарисуйте сами.
  • Напечатайте вектор.
Продвинутый уровень
  • Управляйте своим ЧПУ вручную или с помощью прямых команд
  • Отредактируйте полученный вектор, добавьте на него элементы
  • Создайте макросы, для дальнейшего их размещения на макрос-панели

API и информация для разработчиков

Plotter Control - бесплатная, открытая программа,исходный код которой может скачать каждый с моей MS Foundation страницы здесь или с моего GitHub'а по этой ссылке. Там же вы сможете найти отдельно собранный API - набор библиотек (в подавляющем большинстве моих) необходимых для программирования работы устройства типа ЧПУ. Программа писана на языке C# с небольшими вкраплениями ABC.NET Pascal`я (простите меня уж за это).
